# Demo Script

## Flow

1. Land on homepage → explain SafeCircle in one sentence
2. Sign up as a gig worker (DoorDash driver, no renters insurance)
3. Create a circle → share invite code → second member joins
4. Complete Risk X-Ray questionnaire → show AI analysis loading → reveal risk score
5. Show group dashboard — gap cards, poverty tax meter, unclaimed benefits
6. Switch to Crisis Mode → select "Car Accident" → walk through RED/ORANGE triage steps
7. Show "Don't Sign" warning banner
8. Contribute to emergency pool → show balance update in real-time

## Key Numbers to Mention

- 45M Americans are credit invisible
- $140B in unclaimed benefits annually
- Average poverty tax: $4,500+/year
- One ER visit without insurance: $2,200+
- Filing unemployment 1 week late: $400–600 lost
